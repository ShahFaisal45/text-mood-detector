# 🎭 Mood Detector (Text)

AI-powered mood detection application built with Streamlit and Hugging Face Transformers.

## Features

* Enter any text or sentence
* Detect emotions automatically
* Shows confidence score
* Interactive UI
* Fast predictions using a pre-trained NLP model

## Technologies Used

* Python
* Streamlit
* Hugging Face Transformers
* PyTorch

## Installation

Clone the repository:

```bash
git clone https://github.com/ShahFaisal45/image-mood-detector.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```

## Example Input

```text
I am feeling very happy today!
```

Example Output:

```text
Mood: Joy
Confidence: 94.2%
```
